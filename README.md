## How to Run

1. Clone the repository:
   
       git clone <repository_url>

2. Navigate to the project directory:

       cd flappy-bird-java

3. Compile the code:

       javac *.java

4. Run the game:

       java FlappyBird

## Game Controls

 - Spacebar: Flap the bird's wings to ascend.
 - Avoid the pipes to keep the game running.
